A new homogenized daily maximum and minimum temperature data set, the Australian Climate Observations Reference Network—Surface Air Temperature data set, has been developed for Australia. This data set contains data from 112 locations across Australia, and extends from 1910 to the present, with 60 locations having data for the full post‐1910 period.
