#  *RC* Circuits

An  **RC size 12&#123; ital "RC"&#125; &#123;&#125; circuit** is one containing a  **resistor**<math xmlns:q="http://cnx.rice.edu/qml/1.0" xmlns:m="http://www.w3.org/1998/Math/MathML" xmlns:md="http://cnx.rice.edu/mdml" xmlns="http://cnx.rice.edu/cnxml"><semantics><mrow><mrow><mspace width="0.25em"/><mi>R</mi></mrow><mrow/></mrow><annotation encoding="StarMath 5.0"> size 12&#123;R&#125; &#123;&#125;</annotation></semantics></math> and a  **capacitor**<math xmlns:q="http://cnx.rice.edu/qml/1.0" xmlns:m="http://www.w3.org/1998/Math/MathML" xmlns:md="http://cnx.rice.edu/mdml" xmlns="http://cnx.rice.edu/cnxml"><semantics><mrow><mrow><mi>C</mi></mrow><mrow/></mrow><annotation encoding="StarMath 5.0"> size 12&#123;C&#125; &#123;&#125;</annotation></semantics></math>. The capacitor is an electrical component that stores electric charge.

{'Figure_22_6_1 Figure 22.6.1} shows a simple <math xmlns:m="http://www.w3.org/1998/Math/MathML" xmlns:md="http://cnx.rice.edu/mdml" xmlns="http://cnx.rice.edu/cnxml"><semantics><mrow><mrow><mstyle fontstyle="italic"><mrow><mtext>RC</mtext></mrow></mstyle></mrow><mrow/></mrow><annotation encoding="StarMath 5.0"> size 12&#123; ital &quot;RC&quot;&#125; &#123;&#125;</annotation></semantics></math> circuit that employs a DC (direct current) voltage source. The capacitor is initially uncharged. As soon as the switch is closed, current flows to and from the initially uncharged capacitor. As charge increases on the capacitor plates, there is increasing opposition to the flow of charge by the repulsion of like charges on each plate.

In terms of voltage, this is because voltage across the capacitor is given by <math xmlns:q="http://cnx.rice.edu/qml/1.0" xmlns:m="http://www.w3.org/1998/Math/MathML" xmlns:md="http://cnx.rice.edu/mdml" xmlns="http://cnx.rice.edu/cnxml"><semantics><mrow><mrow><mrow><msub><mi>V</mi><mrow><mtext>c</mtext></mrow></msub><mo stretchy="false">=</mo><mrow><mi>Q</mi><mo stretchy="false">/</mo><mi>C</mi></mrow></mrow></mrow><mrow/></mrow><annotation encoding="StarMath 5.0"> size 12&#123;V rSub &#123; size 8&#123;c&#125; &#125; =Q/C&#125; &#123;&#125;</annotation></semantics></math>, where <math xmlns:q="http://cnx.rice.edu/qml/1.0" xmlns:m="http://www.w3.org/1998/Math/MathML" xmlns:md="http://cnx.rice.edu/mdml" xmlns="http://cnx.rice.edu/cnxml"><semantics><mrow><mrow><mi>Q</mi></mrow><mrow/></mrow><annotation encoding="StarMath 5.0"> size 12&#123;Q&#125; &#123;&#125;</annotation></semantics></math> is the amount of charge stored on each plate and <math xmlns:q="http://cnx.rice.edu/qml/1.0" xmlns:m="http://www.w3.org/1998/Math/MathML" xmlns:md="http://cnx.rice.edu/mdml" xmlns="http://cnx.rice.edu/cnxml"><semantics><mrow><mrow><mi>C</mi></mrow><mrow/></mrow><annotation encoding="StarMath 5.0"> size 12&#123;C&#125; &#123;&#125;</annotation></semantics></math> is the  **capacitance**. This voltage opposes the battery, growing from zero to the maximum emf when fully charged. The current thus decreases from its initial value of <math xmlns:q="http://cnx.rice.edu/qml/1.0" xmlns:m="http://www.w3.org/1998/Math/MathML" xmlns:md="http://cnx.rice.edu/mdml" xmlns="http://cnx.rice.edu/cnxml"><semantics><mrow><mrow><mrow><msub><mi>I</mi><mrow><mn>0</mn></mrow></msub><mo stretchy="false">=</mo><mfrac><mtext>emf</mtext><mi>R</mi></mfrac></mrow></mrow><mrow/></mrow><annotation encoding="StarMath 5.0"> size 12&#123;I rSub &#123; size 8&#123;0&#125; &#125; = &#123;  &#123;&quot;emf&quot;&#125;  over  &#123;R&#125; &#125; &#125; &#123;&#125;</annotation></semantics></math> to zero as the voltage on the capacitor reaches the same value as the emf. When there is no current, there is no <math xmlns:q="http://cnx.rice.edu/qml/1.0" xmlns:m="http://www.w3.org/1998/Math/MathML" xmlns:md="http://cnx.rice.edu/mdml" xmlns="http://cnx.rice.edu/cnxml"><semantics><mrow><mrow><mstyle fontstyle="italic"><mrow><mtext>IR</mtext></mrow></mstyle></mrow><mrow/></mrow><annotation encoding="StarMath 5.0"> size 12&#123; ital &quot;IR&quot;&#125; &#123;&#125;</annotation></semantics></math> drop, and so the voltage on the capacitor must then equal the emf of the voltage source. This can also be explained with Kirchhoff’s second rule (the loop rule), discussed in <link:>, which says that the algebraic sum of changes in potential around any closed loop must be zero.

The initial current is <math xmlns:q="http://cnx.rice.edu/qml/1.0" xmlns:m="http://www.w3.org/1998/Math/MathML" xmlns:md="http://cnx.rice.edu/mdml" xmlns="http://cnx.rice.edu/cnxml"><semantics><mrow><mrow><mrow><msub><mi>I</mi><mrow><mn>0</mn></mrow></msub><mo stretchy="false">=</mo><mfrac><mtext>emf</mtext><mi>R</mi></mfrac></mrow></mrow><mrow/></mrow><annotation encoding="StarMath 5.0"> size 12&#123;I rSub &#123; size 8&#123;0&#125; &#125; = &#123;  &#123;&quot;emf&quot;&#125;  over  &#123;R&#125; &#125; &#125; &#123;&#125;</annotation></semantics></math>, because all of the <math xmlns:q="http://cnx.rice.edu/qml/1.0" xmlns:m="http://www.w3.org/1998/Math/MathML" xmlns:md="http://cnx.rice.edu/mdml" xmlns="http://cnx.rice.edu/cnxml"><semantics><mrow><mrow><mstyle fontstyle="italic"><mrow><mtext>IR</mtext></mrow></mstyle></mrow><mrow/></mrow><annotation encoding="StarMath 5.0"> size 12&#123; ital &quot;IR&quot;&#125; &#123;&#125;</annotation></semantics></math> drop is in the resistance. Therefore, the smaller the resistance, the faster a given capacitor will be charged. Note that the internal resistance of the voltage source is included in <math xmlns:q="http://cnx.rice.edu/qml/1.0" xmlns:m="http://www.w3.org/1998/Math/MathML" xmlns:md="http://cnx.rice.edu/mdml" xmlns="http://cnx.rice.edu/cnxml"><semantics><mrow><mrow><mi>R</mi></mrow><mrow/></mrow><annotation encoding="StarMath 5.0"> size 12&#123;R&#125; &#123;&#125;</annotation></semantics></math>, as are the resistances of the capacitor and the connecting wires. In the flash camera scenario above, when the batteries powering the camera begin to wear out, their internal resistance rises, reducing the current and lengthening the time it takes to get ready for the next flash.
